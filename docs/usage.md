# Usage (MVP)

1. Install dependencies: `pip install -e .`
2. Run sample app: `python examples/mvp_app.py`

Notes
- The MVP uses AdminLTE v3 and Bootstrap 4 via CDN links to produce AdminLTE-compatible layout and styles.
- For interactivity use Dash callbacks. For richer JS behaviours, add assets under `examples/assets` or `src/bs4dash_py/assets`.
